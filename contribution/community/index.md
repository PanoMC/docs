# Community & Support

Need help or want to discuss a new feature? We're here!

## Discord
Our Discord server is the best place for quick questions, real-time discussions, and community support.
- **Join here**: [panomc.com/discord](https://panomc.com/discord)

## GitHub
For bug reports, feature requests, and code contributions, use our GitHub repositories.
- **Main Org**: [github.com/PanoMC](https://github.com/PanoMC)

Don't hesitate to reach out! If you're unsure where a bug belongs or how to start a specific feature, just ask in the `#development` channel on Discord. We highly value collaborator input and are always happy to guide new contributors.
