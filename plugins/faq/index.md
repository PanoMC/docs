# FAQ Plugin

The **FAQ** plugin provides a structured way to manage and display frequently asked questions on your website. It allows you to organize information into categories, making it easy for users to find answers to common queries.

## ğŸ“‹ Features

### 1. Categories
Organize your FAQs into logical groups.
- **Categorization:** Create categories like "General", "Server Rules", "Donations", etc.
- **Display Order:** Control the sequence of categories.
- **Status:** Easily toggle the visibility of an entire category.

### 2. FAQ Management
Manage individual questions and answers.
- **Rich Text Support:** Write detailed answers using the built-in editor.
- **Ordering:** Set a custom display order for each question within its category.
- **Active State:** Enable or disable specific FAQs without deleting them.

### 3. User Interface
- **Clean Layout:** FAQs are typically displayed in an accordion-style interface for better readability.
- **Mobile Friendly:** Fully responsive design ensures FAQs look great on all devices.
