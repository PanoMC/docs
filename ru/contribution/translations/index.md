# Переводы

Помогите нам сделать Pano доступным для всех во всем мире! Переводы — это жизненно важная часть наших усилий по развитию сообщества.

## Текущий статус
На данный момент мы управляем переводами напрямую через наши репозитории GitHub. В будущем мы планируем перейти на специальную платформу для переводов (например, Crowdin или Weblate), чтобы сделать этот процесс еще проще для всех.

**Официально поддерживаемые языки:**
- 🇺🇸 Английский (en)
- 🇹🇷 Турецкий (tr)
- 🇷🇺 Русский (ru)

## Как внести свой вклад
Если вы хотите помочь с переводом Pano или исправить существующие переводы, вы можете отправить Pull Request в соответствующие репозитории, перечисленные ниже.

### Frontend и UI
Эти проекты содержат переводы для веб-интерфейсов:

- **panel-ui**: [github.com/PanoMC/panel-ui/tree/dev/lang](https://github.com/PanoMC/panel-ui/tree/dev/lang)
- **setup-ui**: [github.com/PanoMC/setup-ui/tree/dev/lang](https://github.com/PanoMC/setup-ui/tree/dev/lang)
- **Базовые переводы тем**: общие базовые переводы UI для тем теперь находятся в репозитории движка, в [github.com/PanoMC/sdk/tree/dev/packages/theme-core/lang](https://github.com/PanoMC/sdk/tree/dev/packages/theme-core/lang). Отдельные темы содержат лишь небольшую папку `lang-overrides/`, поэтому базовые строки переводите в репозитории движка.

### Backend и MC Plugin
Переводы для ядра платформы и плагина Minecraft можно найти здесь:

- **Pano (Core и Plugin)**: [github.com/PanoMC/Pano/tree/alpha/Pano/src/main/resources/locales](https://github.com/PanoMC/Pano/tree/alpha/Pano/src/main/resources/locales)
