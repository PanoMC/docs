# Установка Pano

Следуйте этому руководству, чтобы настроить **Pano** и запустить его на своем сервере Minecraft всего за несколько минут.

## ⚙️ Требования

Перед установкой Pano убедитесь, что ваше окружение соответствует следующим требованиям:

1. **Java (JVM 11+)**
   - Pano работает на Java 11 или выше.
   - Убедитесь, что **JDK** или **JRE** установлены и доступны через командную строку.
   - [→ Скачать Java](https://www.oracle.com/java/technologies/javase-downloads.html)

2. **MySQL 5.5+ или MariaDB**
   - Для хранения данных требуется база данных MySQL или MariaDB.
   - Префикс таблиц по умолчанию: `pano_` (вы можете изменить его во время настройки).
   - [→ Руководство по установке MySQL](https://dev.mysql.com/doc/mysql-installation-excerpt/5.7/en/)

3. **Подключение к Интернету**
   - Активное подключение к Интернету **обязательно во время процесса настройки**, так как Pano загружает начальные ресурсы и зависимости.

4. **Открытый порт 80 (HTTP)**
   - Убедитесь, что TCP-порт **80** доступен, если вы устанавливаете Pano на удаленном сервере.
   - **Дополнительно:** Если вы настроите SSL после установки, порт **443 (HTTPS)** также должен быть открыт.

## 📦 Загрузка Pano

Вы можете получить последнюю версию Pano с официального сайта:

- [Скачать последнюю версию →](https://panomc.com/download)
- Для старых версий посетите [GitHub Releases](https://github.com/PanoMC/Pano/releases)

Pano распространяется в виде **`.jar`** файла — точно так же, как **Spigot** или **Paper**.  
После загрузки сохраните файл в удобном месте (например, в специальной папке, такой как `/pano`).

> [!NOTE]
> При первом запуске Pano извлечет необходимые файлы, создаст директории (например, `plugins/` и `themes/`), а также загрузит необходимые зависимости. Настоятельно рекомендуется запускать его в отдельной папке.

## 🚀 Запуск Pano

Чтобы запустить Pano, откройте терминал (или командную строку) и выполните:

```bash
java -jar Pano-<version>.jar
```

Если доступна графическая оболочка, **Pano автоматически запустит графический интерфейс (GUI)**.  
В противном случае работа продолжится в консольном режиме.

### 🖥️ Поведение GUI

- **Двойной щелчок** по `.jar` файлу попытается автоматически запустить GUI.
- Используйте флаг **`-nogui`**, если предпочитаете запускать Pano в консольном режиме:

```bash
  java -jar Pano-<version>.jar -nogui
```

- Встроенный GUI включает минимальную консоль **только для команд Pano**.

Если GUI не может запуститься (например, на сервере без графической оболочки), консоль продолжит работу автоматически.

## 🧭 Мастер настройки (шаг за шагом)

![](/img/installer-view.png)

После запуска Pano откройте браузер и перейдите по адресу:

```
http://<ip-вашего-сервера>/
```

> Убедитесь, что порт **80** открыт и не используется другим приложением.

Вы увидите **Мастер настройки**, который проведет вас через пять простых шагов:

1. **Выбор языка**  
   Выберите предпочтительный язык.  
   <small>*(Продолжая, вы соглашаетесь с Условиями использования и Политикой конфиденциальности Pano.)*</small>

2. **Конфигурация веб-сайта**  
   Настройте данные вашего сайта (название сайта, URL и т. д.).

3. **Настройка базы данных**  
   Введите учетные данные **MySQL** или **MariaDB**.  
   Префикс таблиц по умолчанию: `pano_` (при необходимости его можно изменить).  
   > 💡 **Портативная БД:** В системах **Windows (x64 или ARM64)** вы увидите опцию использования **Портативной базы данных**. При выборе этого варианта Pano автоматически загрузит, настроит и будет управлять портативным экземпляром MariaDB за вас.

4. **Настройки SMTP (необязательно)**  
   Настройте отправку электронной почты, если хотите.  
   Этот шаг является **необязательным**, и если вы решите его пропустить — **не беспокойтесь!**  
   Вы всегда можете включить или обновить настройки SMTP позже в разделе **Панель → Настройки → Платформа**.  
   *(Однако некоторые функции, такие как восстановление пароля, могут не работать до настройки SMTP.)*

5. **Привязка аккаунта и создание администратора**
   - Если у вас уже есть **Pano Account**, вы можете привязать его здесь, нажав кнопку **connect** — ваш e-mail и имя пользователя заполнятся автоматически.
   - В противном случае создайте новый **аккаунт администратора** (не забудьте пароль — он понадобится для входа!).

Чтобы завершить настройку, нажмите **Finish**.  
Pano завершит установку и автоматически перенаправит вас в вашу новую **панель администратора**.

> 🪄 *Теперь вы сможете создавать записи, подключать ваш сервер Minecraft, устанавливать аддоны, менять темы и многое другое!*


## 🎮 Подключение вашего сервера Minecraft (необязательно)

Чтобы включить интеграцию с игрой и подключить ваш сервер Minecraft к Pano, вам потребуется установить **Pano MC Plugin**.

### Что такое Pano MC Plugin?

Pano MC Plugin выступает в качестве моста между вашим сервером Minecraft и Pano, обеспечивая:
- 🔒 Безопасную WebSocket-связь (шифрование RSA + AES-256)
- 🔄 Синхронизацию данных игроков в реальном времени
- 🧩 Автоматическое обнаружение и интеграцию плагинов (AuthMe, права доступа и т. д.)
- 📡 Систему событий для связи игры с веб-сайтом

### Шаги по установке

1. **Скачивание плагина**
   - Посетите страницу [релизов Pano MC Plugin](https://github.com/PanoMC/pano-mc-plugin/releases)
   - Скачайте подходящий JAR-файл для вашей серверной платформы:
     - `pano-spigot-<version>.jar` — Для Spigot/Paper/Folia
     - `pano-bungeecord-<version>.jar` — Для Bungeecord
     - `pano-velocity-<version>.jar` — Для Velocity

2. **Установка плагина**
   - Поместите скачанный JAR-файл в папку `plugins/` вашего сервера
   - Перезапустите сервер Minecraft

3. **Подключение к Pano**
   - После перезапуска откройте **панель администратора Pano**
   - Перейдите в раздел **Серверы** (в боковой панели)
   - Нажмите кнопку **+**, чтобы добавить новый сервер
   - Появится окно подключения с пошаговыми инструкциями
   - Следуйте шагам, показанным в окне, чтобы привязать ваш сервер Minecraft
   - Плагин автоматически установит безопасное зашифрованное соединение

4. **Проверка соединения**
   - После подключения ваш сервер появится в списке **Серверы**
   - Вы должны увидеть статус сервера (онлайн/оффлайн), количество игроков и другую информацию
   - Соединение теперь активно и готово к интеграциям

> 💡 **Примечание:** Вы можете подключить несколько серверов Minecraft к одному экземпляру Pano.
