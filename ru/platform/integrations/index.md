# Интеграции с игрой

Pano поддерживает плавную интеграцию с различными игровыми плагинами для расширения возможностей вашей платформы. Эти интеграции работают по принципу **plug-and-play** — при правильной настройке они автоматически подключаются и синхронизируются с вашим сервером Minecraft.

## 🔌 Pano MC Plugin

Для всех игровых интеграций требуется установка **Pano MC Plugin** и его подключение к вашему экземпляру Pano. Pano MC Plugin выступает в качестве моста между вашим сервером Minecraft и Pano, обеспечивая связь в реальном времени и синхронизацию данных.

**Ключевые особенности:**

- 🔒 **Сквозное шифрование** — Безопасная WebSocket-связь с использованием гибридного шифрования RSA + AES-256
- 🔄 **Синхронизация в реальном времени** — Мгновенная синхронизация данных игроков между игрой и вебом
- 🎮 **Мультиплатформенная поддержка** — Работает с Spigot, Paper, Folia, Bungeecord и Velocity
- 🧩 **Хуки для плагинов** — Автоматически обнаруживает и подключается к поддерживаемым плагинам (AuthMe, права доступа и т. д.)
- 📡 **Система событий** — Слушает игровые события и отправляет их в Pano
- ⚙️ **Автоматическая настройка** — Автоматически корректирует настройки плагинов для обеспечения совместимости

> ⚠️ **Важно:** Всегда используйте **последнюю версию** Pano MC Plugin для обеспечения совместимости со всеми интеграциями и новейшими функциями.

[→ Скачать Pano MC Plugin](https://github.com/PanoMC/pano-mc-plugin/releases) | [→ Руководство по установке](../installation/)

## 📋 Доступные интеграции

### Аутентификация и управление игроками

- [**AuthMeReloaded**](./authme/) — Бесшовная интеграция системы аутентификации
- [**LuckPerms**](./luckperms/) — Полная интеграция с самой продвинутой системой прав доступа в Minecraft
- [**Ban Management**](./ban-management/) — Синхронизированное в реальном времени управление банами между игрой и вебом

## 🎯 Как работают интеграции

Интеграции Pano разработаны для максимально простой работы с поддерживаемыми плагинами:

1. **Установите** поддерживаемый плагин на ваш сервер Minecraft (Spigot/Paper/Folia)
2. **Подключите** ваш сервер Minecraft к Pano с помощью [Pano MC Plugin](../installation/) (через вкладку **Серверы** → кнопка **+** в панели)
3. **Проверьте**, включена ли интеграция в разделе **Панель → Настройки сервера → Интеграция с игрой** (большинство интеграций включены по умолчанию)
4. Вот и всё! Pano автоматически обнаружит плагин и подключится к нему.

## 🔌 Особенности интеграций

Когда интеграция активна, Pano может:

- ✅ Синхронизировать данные игроков между игрой и вебом
- ✅ Обрабатывать процессы аутентификации (вход, регистрация, управление паролями)
- ✅ Выполнять команды плагинов из панели управления
- ✅ Слушать события плагинов и реагировать на них соответствующим образом
- ✅ Автоматически настраивать параметры плагинов для оптимальной совместимости
