# Плагин Avatar

Плагин **Avatar** позволяет каждому зарегистрированному пользователю выбирать источник своего аватара профиля — **голову его скина Minecraft**, его **Gravatar** или **загруженное собственное изображение**. Выбранный аватар затем заменяет стандартную картинку профиля по всему сайту — везде, где отображается изображение пользователя. Администраторы решают, какие источники разрешены, задают ограничения на размер и тип загружаемых файлов и могут изменить аватар любого игрока прямо из панели.

## Возможности

- **Три источника аватара:**
  - **Minotar** — лицо скина Minecraft игрока, загружаемое с `minotar.net`. Это источник по умолчанию для пользователей, которые никогда не выбирали источник.
  - **Gravatar** — общепризнанный аватар, привязанный к email учётной записи пользователя (с запасным вариантом `identicon`).
  - **Custom Upload** — изображение, загруженное пользователем.
- **Оптимизированные загрузки:** принимаются JPG, PNG и GIF (настраивается). Изображения сжимаются и уменьшаются до максимум 256px, тогда как GIF сохраняются как есть, чтобы сохранить анимацию. Старые файлы автоматически удаляются, когда пользователь заменяет изображение или меняет источник.
- **Переопределение администратором:** изменяйте или удаляйте аватар любого игрока из модального окна редактирования игрока в панели.
- **Журнал активности:** изменения настроек и аватаров игроков записываются в журнал активности панели.
- **Локализация:** поставляется с переводами на английский, турецкий и русский.
- **Чистое удаление:** при удалении плагина удаляются его таблица базы данных и все загруженные файлы.

::: tip
Аватары Minotar имеют смысл, только когда имена пользователей на вашем сайте совпадают с реальными аккаунтами Minecraft. Изображения Minotar и Gravatar загружаются браузером каждого посетителя напрямую с `minotar.net` / `gravatar.com`, поэтому эти сервисы должны быть доступны для ваших посетителей.
:::

## Для ваших пользователей

После включения плагина авторизованные пользователи находят раздел **Profile Avatar Settings** в верхней части страницы **Settings** своего аккаунта. Там они могут выбрать разрешённый источник, увидеть предпросмотр в реальном времени и — для собственных загрузок — загрузить, просмотреть или удалить изображение. Показываются только те источники, которые вы разрешили.

## Настройки администратора

- **Настройки плагина:** откройте **Panel → Plugins → Avatar**. Карточка **Avatar Settings** позволяет задать **Max Upload Size** (в МБ), **Allowed Image Types** (PNG / JPEG / GIF) и **Allowed Avatar Sources** (Minotar / Gravatar / Custom Upload).
- **Редактирование аватара игрока:** откройте страницу деталей игрока и используйте строку **Profile Avatar** в модальном окне редактирования, чтобы изменить или удалить его картинку.

## Необходимые права доступа

- **Manage Avatar Settings** — настройка разрешённых источников, типов файлов и ограничений на размер.
- **Manage Player Avatar** — изменение аватаров профилей игроков со страницы деталей игрока.

::: warning
Конструктор **Custom URL Sources** на карточке настроек ещё не работает. Вы можете настроить записи, но пока их ничто не использует — они не появятся как выбираемый источник аватара. Пока оставьте этот раздел пустым.
:::

## Настройка

1. Включите плагин в **панели администратора Pano**.
2. Перейдите в раздел **Panel → Plugins → Avatar**.
3. Настройте ваши **Avatar Settings** (разрешённые источники, типы изображений и максимальный размер загрузки).
4. Теперь пользователи могут выбирать свой аватар со страницы **Settings** своего аккаунта!
